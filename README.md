# 🎵 Spotify Data Analysis using Python

## 📖 Project Overview
This project focuses on analyzing Spotify music data to understand trends, patterns, and insights related to songs, artists, and audio features. The analysis is performed using Python by cleaning the dataset, exploring key attributes, and visualizing patterns in music data.

The goal is to uncover insights such as popular genres, artist trends, and relationships between audio features like energy, danceability, and popularity.

---

## 🎯 Objective
- To analyze Spotify dataset and understand music trends  
- To explore relationships between different audio features  
- To identify patterns in song popularity  
- To visualize insights using charts and graphs  

---

## 🛠️ Tools & Libraries

- Seaborn
